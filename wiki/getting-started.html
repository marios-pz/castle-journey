Getting Started

Controls

  • Movement: Use WASD or arrow keys to move your character
  • Menu: Press ESC to open/close the pause menu
  • Inventory: Press E or I to open/close your inventory

Basic Mechanics

Gold System

Gold is the main currency in Castle Journey. You start with 50 gold and can earn more by completing quests and exploring the world.

Quest System

Quests are the main way to progress through the game. Each quest has:

  • A title and description
  • Specific objectives to complete
  • Gold rewards upon completion

Navigation

The game features multiple levels that you can navigate between. Use the navigation buttons in each level to move between areas.
